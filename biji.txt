工作区        暂存区	    仓库

初始化仓库
1、直接远程克隆（空文件夹，克隆下来得项目直接就是仓库）
（git clone git@github.com:junbin721/gitbranch.git）

2.、git init（自己初始化仓库，是否为空文件夹都可以）


提交内容到仓库
1. git add *  	内容提交到了暂存区
（git add *.html git add /test/      git add biji.txt）
2. git commit -m message   提交到仓库
  会生成commit
配合一些指令做操作：
git status    查看状态 
git checkout -- biji.txt    撤销工作区的修改
git restore biji.txt  撤销工作区修改git
git restore --staged biji.txt   撤销暂存区修改
git log   打印全部的日志信息
git log -1  打印最近一次提交   -2  最近两次
git log --pretty=oneline   打印一行
git log --oneline  打印一行
git log --graph --pretty=oneline --abbrev-commit  打印详细的带分支的信息
git config --global alias.lg 'log --graph --pretty=oneline --abbrev-commit'    做一些全局的配置别名，可以简化git操作单词长度
git tag -a v1.0 -m 'message说明' commitID   打标签
